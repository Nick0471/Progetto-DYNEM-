🛠️ Stack Tecnologico

Il nostro "AI Driver" è stato costruito utilizzando il seguente stack ufficiale della competizione:

Piattaforma/Ambiente: TORCS (The Open Racing Car Simulator).

Linguaggio di Programmazione: Python.

Motore IA / Core Engine: Modelli IA IBM Granite (Open source via IBM SkillsBuild).

IDE di Sviluppo: Visual Studio Code (VSCode).

🏁 Dettagli del Progetto e Regole Rispettate

Circuito: Il codice è ottimizzato esclusivamente per il circuito Corkscrew.

Partenza: L'algoritmo gestisce una partenza da fermo (Standing Start).

Integrità del Gioco: È stato modificato solo il codice Python dell'IA, senza alterare le dinamiche interne di TORCS.

Design: Abbiamo implementato una livrea personalizzata per le gare di massa, inviata separatamente come richiesto.
